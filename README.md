# Credit_Card_Financial_Dashboard
Engineered a comprehensive Financial Dashboard using Power BI and PostgreSQL to analyze real-time credit card transaction data Streamlined the ETL pipeline by resolving schema inconsistencies and optimizing SQL queries for efficient data extraction. Implemented complex DAX measures to track critical KPIs (Revenue, Interest, Delinquency), delivering a robust analysis tool that provides stakeholders with actionable insights for strategic decision-making.
